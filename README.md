# WLED CLI

Simple *stateless* CLI for WLED instances.

## Install

Build it from sources or install binary from [release page](https://github.com/dector/wled-cli/releases/latest).

## Features

### Toggle power

```
./wled toggle <IP>
```

For example:

```
./wled toggle 192.168.1.21
```

### Check all commands

```
./wled help
```
